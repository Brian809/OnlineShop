# OnlineShop 商城应用开发路线图

## 项目概述
构建一个简单的B2C电商平台，包括商品展示、购物车、订单管理和用户认证等核心功能。

## 技术栈

### 后端
- **框架**: Express.js
- **认证**: Passport.js
- **ORM**: Sequelize
- **数据库**: MySQL / PostgreSQL
- **密码加密**: bcrypt
- **验证**: joi / express-validator
- **日志**: morgan / winston

### 前端
- **框架**: Vue 3
- **构建工具**: Vite
- **路由**: Vue Router
- **状态管理**: Pinia
- **HTTP 客户端**: Axios
- **样式**: CSS3 / TailwindCSS (可选)

### 工具
- **版本控制**: Git
- **包管理**: npm / yarn
- **开发**: VS Code
- **数据库管理**: MySQL Workbench / pgAdmin
